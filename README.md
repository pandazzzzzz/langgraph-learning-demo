# LangGraph Learning Demo

🚀 一个完整的LangGraph学习项目，从基础概念到高级多智能体系统的渐进式学习路径。

## 📖 项目简介

本项目是基于LangGraph框架的综合学习系统，旨在帮助开发者系统性地掌握AI Agent开发技能。项目采用模块化设计，提供由浅入深的学习路径，涵盖状态管理、工具集成、RAG系统和多智能体协作等核心概念。

## 🚀 快速开始

```bash
# 克隆项目
git clone <your-repo-url>
cd langgraph-learning-demo

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑.env文件，添加你的OpenAI API密钥

# 运行第一个示例
python src/examples/basic_chatbot.py
```
